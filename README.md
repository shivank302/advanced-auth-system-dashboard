# 🔐 Advanced Auth System Dashboard

A modern and interactive **Authentication System with Admin Dashboard** built using **HTML, CSS, and JavaScript**.

This project demonstrates a complete frontend-based user management system with advanced UI features and localStorage integration.

---

## 🌐 Live Demo

🚀 https://shivank302.github.io/advanced-auth-system-dashboard/ 

---

## ✨ Features

* 🔑 Multi-Step Registration Form
* 📊 Progress Bar Indicator
* 🔒 Password Strength Checker
* 🌙 Dark Mode Toggle
* 🧑‍💻 Admin Dashboard
* 🔄 User Status Toggle (Active/Inactive)
* ✏️ Edit & 🗑 Delete Users
* 🔍 Search Functionality
* 💾 Data Stored in LocalStorage
